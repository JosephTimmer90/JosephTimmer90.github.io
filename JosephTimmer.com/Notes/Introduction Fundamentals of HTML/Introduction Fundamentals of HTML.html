The Body

Only content inside the opening and closing body tags can be displayed to the screen.

HTML Structure

Parent and child relationship of tags.

Headings

There are six headings. h1 is the largest and h6 is the smallest.

Divs

Divs are used for grouping elements together.

Attributes

Attributes are added to the opening tag. Attributes are made up of a name and a value.

Displaying Text

Paragraphs contain a block of plain text.

Span separates small pieces of content which are on the same line as other content.

Styling Text

  • The em tag emphasises text and generaly renders in italics.
  • The strong tag highlights important text and generally renders as bold.

Line Breaks

The line break element is only composed of a starting tag.
You can use it anywhere in your code and a line break will be shown in your browser.



Unordered Lists

  • The unordered list tag (ul) creates a list of items in no particular order.
  • The ul tag outlines list items with bullet points.
  • Each item in the list needs an li tag.

Ordered Lists

  1. Orderd lists create numbered lists.
  2. Ordered lists are usefull for listing steps or ranking items.
  3. Each item in the list needs an li tag.

Images

  • The image tag (img) is a self closing tag.
  • Self-closing tags may include or omit the final slash.
  • The img tag has a required attribute called src.
  • The src must be the uniform resource locator (URL) of the image.

Image Alts

  • The alt attribute brings meaning to images on our sites.
  • Can be added to the image tag just like src attribute.
  • If an image fails to load on a web page, a user can mouse over the area originally intended for the image and read a brief description of the image. This is made possible by the description you provide in the alt attribute
  • Visually impaired users often browse the web with the aid of screen reading software. When you include the alt attribute, the screen reading software can read the image’s description out loud to the visually impaired user.
  • The alt attribute also plays a role in Search Engine Optimization (SEO), because search engines cannot “see” the images on websites as they crawl the internet. Having descriptive alt attributes can improve the ranking of your site.
  • The alt attribute should be left blank for images that do not convey meaningful information.

Videos

  • The video element requires a src attribute with a link to the video source.
  • Requires opening and closing tags.
  • Width and height attributes are self explanatory.
  • The controls attribute instructs the browser to include basic video controls.
  • The text between the opening and closing tags will only be displayed if the video is unable to load.
