Main and Footer

  • The main tag holds the dominant content of the webpage.
  • The footer tag holds:
    • Contact information
    • Copyright information
    • Terms of use
    • Site Map
    • Reference to top of page links

Article and Section

  • The section tag splits a webpage into separate sections.
  • The article tag holds content that makes sense on its own.

The Aside Element

  • The aside element is used to mark additional information that can enhance abother element but isn't required in order to understand the main content. Examples are:
    • Bibliographies
    • Endnotes
    • Comments
    • Pull quotes
    • Editorial sidebars
    • Additinal information
    • The aside tage could be used for something like this sentence.

Figure and Figcaption

  • figure is an element used to encapsulate media such as an image, illustration, diagram, code snippet, etc, which is referenced in the main flow of the document.
  • figcaption is an element used to describe the media in the figure tag.
  • Usually figcaption will go inside the figure tag, so it will stay with the figure when moved.

Audio and Attributes

  • THe controls attribute automatically displays the audio controls into the browser such as play and mute.
  • the src attribute specifies the URL of the audio file.

Video and Embed

  • controls attribute provides controls
  • auto play plays the video on page load
  • loop loops the video
  • embed is deprecated

Review

  • Semantic HTML introduces meaning to a page through specific elements that provide context as to what is in between the tags.
  • Semantic HTML is a modern standard and makes a website accessible for people who use screen readers to translate the webpage and improves your websiteâ€™s SEO.
  • header tag, nav tag, main tag, and footer tag create the basic structure of the webpage.
  • The section tag defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
  • The article tag holds content that makes sense on its own such as articles, blogs, comments, etc.
  • The aside tag contains information that is related to the main content, but not required in order to understand the dominant information.
  • The figure tag encapsulates all types of media.
  • The figcaption tag is used to describe the media in the figure tag.
  • The video tag, embed tag, and audio tag elements are used for media files.
