Introduction

  • The command line is a text interface that takes commands and passes them on to the computer's operating system to run.
  • With the command line you can run programs and write scripts to automate common tasks.
  • This course is for unix-based systemns.
  • When using the command line we refer to folders as directories. Files and directories on your computer are organized into a filesystem.

Filesystem

  • The first directory in the file system is the root directory.

ls

  • $ is called the shell prompt.
  • ls lists all of the files and directories for the directory you are in.
