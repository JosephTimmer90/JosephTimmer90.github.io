Table Headings

  • The th tag is for table headings.
  • The th tag functions just like the td tag except that it is for relevant titles.
  • The th tag has a scope attribute which can take one of two values.
    • row - making it clearly a row heading.
    • col - making it clearly a column heading.

Table Body

The tbody tag can be used to section off all of the tables data.

Table Head

The thead tag can be used to section off the column headers from the rest of the table.

Table Footer

The tfoot tag can be used to section off the bottom part of a long table.
Footers are often used to contain sums, differences, and other data results.
