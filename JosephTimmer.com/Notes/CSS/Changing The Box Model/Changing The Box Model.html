Changing The Box Model

Why Change the Box Model?

Adding all elements of the box model together can make sizing content difficult.

Box Model: Content-Box

  • The box-sizing property controls the type of box model the browser uses.
  • content-box is the default property.

Box Model: Border-Box

  • The border-box model works from outside in. The content box works from inside out.
  • In border-box model the height and width remain fixed and the border thickness and padding will be included inside the box.

The New Box Model

Use the universal selector to change the box model for all elements.

Review: Changing the Box Model

  • In the default box model, box dimensions are affected by border thickness and padding.
  • The box-sizing property controls the box model used by the browser.
  • The default value of the box-sizing property is content-box.
  • The value for the new box model is border-box.
  • The border-box model is not affected by border thickness or padding.
