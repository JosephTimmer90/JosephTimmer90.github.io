Selectors

Selectors

Type

  • A selector is used to target specific HTML elements to be styled by the declaration.
  • The type selector does not include the angle brackets.
  • The type selector is sometimes referred to as the tag name or element selector.

Universal

  • The universal selector selects all elements of any type.
  • * is the universal selector.

Class

  • A class attribute is one of the most common ways to select an element.
  • To select an HTML element by its class, a period must be prepended to the class's name.

This section has an id of multiple classes.

Multiple Classes

  • It is possible to add more than one class name to an HTML elements class attribute.
  • Multiple classes are added by separating them with a space.

ID

  • ID can only have a single value and only be used once per page.
  • To select an element's ID, prepend the ID name with a #.

Attribute

  • Attributes like href and src can be selected. This eliminates the need to add classes or ids.
Home.

Pseudo-class

  • The apperance of certain elements can change or be in a different state after certain user interactions.
  • :focus, :visited, :disabled, and :active are all pseudo-classes.
  • A pseudo class can be attached to any selector. It is written as selector:pseudo-class.

Classes and IDs

  • Classes are meant to be reused over many elements.
  • An ID is meant to style only one element.

Specificity

  • Specificity is the order by which the browser decides which CSS styles will be displayed.
  • Best practice is to use the lowest level of specificity, so it can be over ridden later.
  • Order from most specific to least specific is as follows.
    1. IDs
    2. Classes
    3. Type

Chaining

  • It is possible to require that an HTML element has two or more CSS selectors at the same time.
  • This is refered to as chaining.

Descendant Combinator

  • Nested elements(descendants) can be selected.
  • The format is ParentElement descendant {}

Chaining and Specificity

  • Chaining makes selectors more specific.

Multiple Selectors

Combine selectors with the same styling by using a coma.
This prevents the need to write the same style twice.

Review

  • CSS can select HTML elements by type, class, ID, and attribute.
  • All elements can be selected using the universal selector (*).
  • An element can have different states using the pseudo-class selector (:).
  • Multiple CSS classes can be applied to one HTML element.
  • Classes can be reusable, while IDs can only be used once.
  • Most specific to least specific.
    • ID
    • Class
    • Type
  • Multiple selectors can be chained together.
  • Nested elements can be selected by separating selectors with a space.
  • Multiple unrelated selectors can recieve the same styles by separating the selector names with commas.
